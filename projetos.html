Cadastro MVC com PHP

Este é um projeto de cadastro de usuários implementado utilizando PHP e a arquitetura MVC. O sistema permite criar, listar, editar e excluir cadastros de usuários, com comunicação com o banco de dados MariaDB via PDO.
