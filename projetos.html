PHP MVC - Sistema de Login Seguro com Painel Administrativo

Este projeto implementa um sistema de login seguro utilizando PHP com a arquitetura MVC. Ele oferece um painel administrativo protegido, onde é possível gerenciar usuários e visualizar mensagens recebidas via formulário de contato. A autenticação é realizada de forma segura com hash de senhas e controle de sessões. Além disso, o sistema inclui proteção contra CSRF, validação de e-mail, e outros mecanismos de segurança avançados.

O painel inclui funcionalidades como:

  • Login seguro com hash de senha
  • Gestão de usuários cadastrados
  • Visualização e gerenciamento das mensagens recebidas via formulário de contato
  • Dashboard com estatísticas de usuários e mensagens
  • Proteção de rotas com middleware
  • Segurança aprimorada com CSRF tokens e validação rigorosa de dados

Este projeto é uma excelente prática de segurança em PHP, manejo de sessões, CRUDs, SQL e autenticação de usuários.
